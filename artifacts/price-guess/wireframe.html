문제 — 답하기 전

연속 정답 3
문제 #4
배추
1포기 가격은?
Notes

핵심 플레이 화면. 보기 4개는 서로 겹치지 않는 가격 구간이며 정확히 1개만 정답.

  • 트리거: "게임 시작" 또는 직전 문제 정답 후 "다음 문제"
  • 인터랙션: 보기 1개 선택 → 정답이면 정답 공개 상태, 오답이면 오답 공개 상태
  • 전환: 정답 → 문제(정답) / 오답 → 문제(오답)

연속 정답 4
문제 #4
정답!
① 1,000원 ~ 2,000원
② 2,000원 ~ 4,000원
③ 4,000원 ~ 6,000원
④ 6,000원 ~ 8,000원
실제 가격
배추 1포기 3,200원
서울 도매 · 2026년 5월 21일 기준
Notes

정답 시 같은 문제 화면에서 정답 강조 + 실제 가격/출처 패널을 노출한 상태. 연속 정답 +1.

  • 트리거: 답 전 화면에서 정답 보기 선택
  • 인터랙션: 선택한 정답 보기 강조, 실제 가격·기준일·시장 표시 / "다음 문제" 클릭
  • 전환: "다음 문제" → 새 문제(답 전). 캐시로 진행 시(scenario-6) 기준일이 캐시 날짜로 표기됨

문제 — 오답 공개

연속 정답 3
문제 #4
오답!
① 1,000원 ~ 2,000원
② 2,000원 ~ 4,000원
③ 4,000원 ~ 6,000원
④ 6,000원 ~ 8,000원
실제 가격
배추 1포기 3,200원
서울 도매 · 2026년 5월 21일 기준
Notes

오답 시 내가 고른 보기(오답)와 실제 정답 보기를 함께 표시. 연속 정답 더는 오르지 않고 게임 종료 수순.

  • 트리거: 답 전 화면에서 오답 보기 선택
  • 인터랙션: 내 선택(오답)과 정답 동시 표시, 실제 가격 노출 / "결과 보기" 클릭
  • 전환: "결과 보기" → 결과·등록 화면 (최종 연속 정답 = 오답 직전까지 맞힌 수)

결과 · 랭킹 등록

최종 연속 정답
3

닉네임을 입력하세요 (빈 입력 시 표시)

Notes

게임 종료 후 최종 연속 정답 수를 보여주고 온라인 랭킹 등록을 받는 화면.

  • 트리거: 오답 공개 화면에서 "결과 보기"
  • 인터랙션: 닉네임 입력 후 "등록" → 랭킹 반영(빈 입력 시 안내, 중복 등록 방지) / "다시 하기" → 새 게임 / "랭킹 보기" → 랭킹 화면
  • 전환: 등록 완료 → 랭킹 화면(내 항목 강조) / "다시 하기" → 문제(답 전)

  • 전환: 재조회 성공 → 문제(답 전), 실패 → 이 화면 유지
